# Проект: Путешествие по России
###### _Выполнено в процессе обучения по профилю Веб-разработчик в [Яндекс Практикум](https://www.praktikum.yandex.ru)_

**Описание**
Данная практическая работа была выполнена в рамках четвертого спринта. Темы спринта были преимущественно посвящены адаптивной и резиновой вертске, grid'ам, различным особенностям в верстке для разных устройств. Работа адаптирована под разные устройства: 1280px, 1024, 768px и 320px.
